# Cloud Sovereignty Self-Assessment

Single-page tool that scores a cloud provider against the European Commission's
**Cloud Sovereignty Framework**: the overall **SEAL** level (0–4, set by the weakest
objective) and a weighted **Sovereignty Score** out of 100. Bilingual (EN/EL). No
server, no build, no dependencies — open `index.html` or use the live link.

**Live:** https://defstat.github.io/cloud-sov-assessment/

## Tabs

- **Quick** — rate each of the 8 objectives at one SEAL level. Fast, indicative; the
  per-objective score uses a curve *derived* from the calculator's values (not the
  official method).
- **Detailed** — faithful reproduction of the Commission's calculator across its 48
  specific objectives; each answer carries the official value and SEAL tag. Reproduces
  the guidance's worked Sovereignty Score of 68%.

Category weights are the official ones (20/10/10/15/10/15/15/5), editable on the Quick tab.
